    • le premier élément correspond à l'URL de votre vidéo (youtube ou autre)
    • le second à l'image d'aperçu
    • le troisième
    • une quatrième valeur peut être ajouter url : permet de forcer l'affichage de la vidéo en plein écran

List pages


{{ pages_list('exampleTag')|unprose

Discover more on Advanced Pages Listing.

  • max nombre maximum d'items à montrer, peut être un tableau pour paginer le résultat, défault : 0, no limit
  • order default publishedAt DESC,weight DESC
  • view default /component/pages_list.html.twig, accepte tout chemin vers un template ou card pour /component/pages_list_card.html.twig
  • host default current host, if set to '', will be set to current host

Redirection

Votre contenu doit contenir uniquement :

Location: http://example.eg/

Show More

Collapse long content blocks with expand/collapse functionality.


{{ startShowMore() }}

Your long content here (only first 250px height will be visible base on your templete file `/component/show_more.html.twig`)...

{{ endShowMore() }}

# With custom background (for non-white backgrounds)
{{ startShowMore() }}
...
{{ endShowMore('via-gray-100 to-gray-100') }}

# With custom ID (for linking via #hash)
{{ startShowMore('my-custom-id') }}
...
{{ endShowMore() }}
